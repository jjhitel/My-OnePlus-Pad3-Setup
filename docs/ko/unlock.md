# 부트로더 잠금 해제

> **⚠️ 중요한 FRP 경고**
> 최근 OOS 업데이트(Pad 3의 `.701` 버전 등)에서 공장 초기화 후 계정 로그인 시도 자체가 차단되는 버그가 있습니다. **이를 방지하기 위해 먼저 반드시 기기에서 구글 계정을 로그아웃하세요.**

### **드라이버, ADB & Fastboot 설치**

* [설치 프로그램 다운로드](https://github.com/K3V1991/ADB-and-FastbootPlusPlus)

1.  **개발자 옵션 활성화:** `설정 > 기기 정보 > 버전`으로 이동하여 `빌드 번호`를 여러 번 탭하여 확인 메시지가 나올 때까지 누릅니다.

2.  **OEM 잠금 해제 활성화:** `설정 > 시스템 및 업데이트 > 개발자 옵션`으로 이동하여 `OEM 잠금 해제`를 켭니다.

3.  **Fastboot 모드 진입:** 기기 전원을 끕니다. 그 후 `볼륨 감소 + 전원` 버튼을 동시에 길게 눌러 Fastboot 모드로 진입합니다.

4.  **PC 연결:** USB 케이블로 기기를 컴퓨터에 연결합니다.

5.  **잠금 해제 명령어:** 터미널이나 명령 프롬프트를 열고 다음 명령어를 입력합니다:

    ```
    fastboot flashing unlock
    ```

6.  **기기에서 확인:** 기기의 볼륨 키로 `UNLOCK THE BOOTLOADER`를 선택하고 전원 키를 눌러 승인합니다.
